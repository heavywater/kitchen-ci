CSS

Maximum uses in total 15 CSS files. Most pages uses STYLE.css and the bootstrap configuration files. Other CSS are used for specific pages or needs.

This theme was created using Twitter Bootstrap 2.3.1 version. I tried to keep all major configurations by default. The style.css file adds the main configuration and some tweaks in the navigation bar and in the Media Queries.

JavaScript

Maximum uses various JS to work with all the theme. Here I will detail some scripts that need a specific configuration.

 

Highcharts.js / Other-charts.js

Highcharts script is used in the skills.html file to create the donut charts. Other-charts.js contains the specific configuration fot the donut charts.

To modify the values of the charts, open other-charts.js and locate the proper section for each chart. Example:

 

In the image above you see the details of the first chart (used in the skills.html file). You can modify some aspects of the chart, the most important are:

1 - This is the name of the chart. You should reference to this name in your HTML to render the chart.

2 - Change the border size and color of the donut chart.

3 - Here you define the sections of the chart as well the size, color and name of each section.

Please, see the demo page to obtain a deeper understanding. If you need more information, please visit the official script page.

 

prettyPhoto.js

 

The theme uses the pretty photo script to show the projects images.
 
If you need examples of usage or more information about the script, please visit this page.
 
Flexslider.js
 
Is a beautiful slider created by Woothemes. Flexslider is used in some pages of this theme. Bootstrap comes with their own carousel script, but I decided to add flexslider because you can configure more advanced options with this script.
 
You can see a deep configuration and examples of the script here: WooThemes.
 
Timeline.js
 
Timeline.js is not included in the theme but is used in the Timeline.html file. This is a great tool to create your timline. You can create yours using a XLS file or a google doc file. You create all the timeline online and then paste the code in your html (timeline.html uses this option).
 
Also, you can download and modify all the aspect of the file. I strongly recommend that you read carefully the online documentation provided by the creator and use their page to create your timeline. If you feel confident with the script, then download it and make your desired changes.

Page Configuration

SPECIFIC CONFIGURATIONS

Twitter Widget

To show your tweets, you should modify this section located in the footer of each page.

Change the username (green highlighted) with yours. If you want to show more tha 1 tweet, modify the last value of the line, example: "count=3".

 

Google Fonts

Maximum uses 2 fonts in the whole theme, Open Sans and Open Sans Condensed.

 

If you want to change this value, you have to replace all the references in the main.css file and in the html page.
 
In Google Fonts website you can find the code to paste in the theme. Choose the desired font, click in the "Quick Use" link and copy & paste the code.
 

PSD Files

 

Please note: This theme NOT includes a full layered PSD.
 
The theme comes with the demo images, so you can easily modify them with your content.
